# approve-pull-request-action

A GitHub Action for approving pull requests.

## Usage

```yaml
steps:
  - name: Approve Pull Request
    uses: juliangruber/approve-pull-request-action@v1
    with:
      github-token: ${{ secrets.GITHUB_TOKEN }}
      number: 1
```
